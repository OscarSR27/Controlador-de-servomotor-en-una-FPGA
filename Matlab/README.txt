Instrucciones para el uso de los programas de Matlab

1. Utilice el programa Rampa.m para generar el estímulo de entrada para el I_PD implementado en Matlab. La salida de este programa será el archivo FuncRampa.txt

2. Utilice el programa ServoMotor.m para simular el I_PD hecho en Matlab. Además este archivo genera los estímulos para la entrada del I_PD implementado en lenguaje Verilog. Las salidas de este programa son los archivos: Salida.txt, Referencia.txt, y.txt, proporcional.txt, integrador.txt, derivador.txt. Además los archivos que serán utilizados como estímulo son: Referencia_Binario.txt y y_Binario.txt.

3. Utilice el programa BinarioDecimal.m para convertir los valores de binario procedentes de la simulación del I_PD implementado en Verilog a base 10.
